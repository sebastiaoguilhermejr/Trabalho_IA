Arquivos principais
------------------
- src/TSP.py  — Backend do TSP (geração de matriz de distâncias, inicializadores e algoritmos: subida de encosta, têmpera simulada, algoritmo genético, etc.).
- src/Front.py — Interface gráfica (classe `TSPInterface`) para gerar o problema, criar solução inicial, selecionar e executar métodos, e visualizar resultados.
- requirements.txt — Dependências Python necessárias.

Instalar dependências
---------------------
Abra um terminal (bash) na raiz do projeto e execute:

```bash
python -m pip install -r requirements.txt
```
Testes rápidos
--------------
Para verificar que os módulos importam corretamente, execute (na raiz do projeto):

```bash
python -c "import sys; 
sys.path.append('src'); 
import Front, TSP; 
print('Import OK:', TSP.TSP(30).num_cidades)"
```

Executar a interface gráfica
----------------------------
Na raiz do projeto (onde está este arquivo), execute:

```bash
# opção A: rodar a partir da pasta raiz
python src/Front.py

# opção B: entrar na pasta src e rodar
cd src
python Front.py
```
Observações importantes
----------------------
- O TSP exige pelo menos 30 cidades; se você informar um número menor, a interface exibirá um erro.
